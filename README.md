# Tampermonkey scripts

Небольшая коллекция пользовательских скриптов для Tampermonkey.

## Скрипты

- `chatgpt-close-popup.user.js` - закрывает отдельное уведомление на ChatGPT.
- `fbfind-cleaner.user.js` - убирает рекламу, Telegram-промо и лишние вставки на fbfind/fbphdplay.
- `kinokino-button.user.js` - добавляет кнопку перехода на KinoKino со страниц Кинопоиска.

## Установка

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/).
2. Откройте нужный `.user.js` файл на GitHub.
3. Нажмите кнопку `Raw`.
4. Tampermonkey предложит установить скрипт.

После публикации репозитория прямые ссылки будут выглядеть так:

```text
https://raw.githubusercontent.com/Horndayel/tampermonkey-scripts/main/chatgpt-close-popup.user.js
https://raw.githubusercontent.com/Horndayel/tampermonkey-scripts/main/fbfind-cleaner.user.js
https://raw.githubusercontent.com/Horndayel/tampermonkey-scripts/main/kinokino-button.user.js
```

## Обновления

Если скрипт установлен из `raw.githubusercontent.com`, Tampermonkey сможет проверять обновления при изменении версии в заголовке `@version`.
